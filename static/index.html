Developer API

Integrate multi-voice expressive speech generation into your web apps, systems, or games.

Get Started in Minutes

Execute narration generations using simple HTTP POST requests. Here is the easiest way to generate audio using standard shell tools.

# Generate multi-voice narration for your story text curl -X POST https://api.voiceweave.ai/api/narrate \ -H "Content-Type: application/json" \ -d '{ "text": "\"Hello!\" she said. \"Hey there,\" he replied.", "voice_map": { "narrator": "davis", "she": "aria", "he": "guy" } }'

API Resource Endpoints

Explore the endpoints supported by the VoiceWeave API service.

Method Endpoint Description
POST /api/narrate Generate narration from text. Accepts a voice_map.
POST /api/parse Analyze dialogue blocks & characters from text.
POST /api/upload Upload manuscript (.epub, .pdf, .txt) for async generation.
GET /api/voices Get the listing of all available voices.
GET /api/status/:id Poll status for background file conversions.

Authentication

To use our hosted API servers in production, you must register a developer account in your dashboard and request an API key. For local testing during this hackathon, no API token header authentication is active.

Code Integration Examples

import requests payload = { "text": '"No!" she stammered. "Run!" he hissed.', "voice_map": { "narrator": "davis", "she": "aria", "he": "ryan" } } response = requests.post( "https://api.voiceweave.ai/api/narrate", json=payload ) result = response.json() print(f"Audio URL: {result['audio_url']}")
const response = await fetch('/api/narrate', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ text: '"Is anyone there?" she whispered.', voice_map: { narrator: 'davis', she: 'jenny' } }) }); const data = await response.json(); console.log('Generated Audio URL:', data.audio_url);
